总结

case when具有两种形式：

  1. 简单case函数

    CASE sex 

    WHEN '1' THEN '男' 
WHEN '2' THEN '女'

    ELSE ‘其他’ END

    2.case搜索函数

    CASE WHEN sex = ‘1’ THEN ‘男’ 

    WHEN sex = '2' THEN '女'

    ELSE ‘其他’ END
